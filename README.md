Dagim Faji Portfolio Website
Overview
This is a personal portfolio website for Dagim Faji, designed to showcase professional information and a static gallery of images. Features

Responsive Design: Adapts to various screen sizes using Tailwind CSS.
Static Image Gallery: Displays manually added images in a grid layout with hover effects.
Navigation: Includes sections for Home, Gallery, About, and Contact.
Professional Information: Highlights Dagim's IT background and contact details.

Setup Instructions

Clone or Download:
Download the project files or clone the repository to your local machine.


File Structure:
The main file is index.html, which contains the HTML, CSS (via Tailwind CDN), and minimal JavaScript.
Create an images/ folder in the same directory as index.html to store your image files.


Add Images:
Place your images in the images/ folder (e.g., images/sample1.jpg, images/sample2.jpg).
Update the <img> tags in the #galleryGrid section of index.html to reference your images (e.g., <img src="images/your-image.jpg" alt="Description">).
Add more <div> elements with <img> tags in the #galleryGrid to include additional images.


Run the Site:
Open index.html in a web browser (e.g., Chrome, Firefox) to view the site locally.
Alternatively, serve the file using a local server (e.g., npx live-server or Python’s http.server) for a better development experience.


Customization:
Update the text in the About and Contact sections to reflect your personal details.
Modify colors and styles by adjusting Tailwind classes or adding custom CSS in the <style> tag.



Deployment

Static Hosting:
Deploy the site to platforms like GitHub Pages, Netlify, or Vercel by uploading the index.html file and the images/ folder.
Ensure the images/ folder is included in the deployment to make images accessible.


File Paths:
The image paths in index.html (e.g., images/sample1.jpg) are relative. Ensure your hosting platform preserves the folder structure.



Technologies Used

HTML5: Structure of the website.
Tailwind CSS: Styling and responsive design (loaded via CDN).
JavaScript: Minimal usage for basic functionality.

Notes

Ensure an internet connection to load Tailwind CSS from the CDN. For offline use, download Tailwind CSS and host it locally.
Use high-quality, optimized images (e.g., JPEG or WebP format) to ensure fast loading times.
Tested on modern browsers (Chrome, Firefox, Edge).
